.. _emconf-defaultCaptchaProvider:

Default Captcha Provider
""""""""""""""""""""""""

.. container:: table-row

   Property
       defaultCaptchaProvider

   Data type
       string

   Description
       Shows a selection of available captchaProviders. Choose one to use as
       default.

       .. tip::

           Developers who might want to implement their own captcha should refer to
           :ref:`section-developer-captcha`.
